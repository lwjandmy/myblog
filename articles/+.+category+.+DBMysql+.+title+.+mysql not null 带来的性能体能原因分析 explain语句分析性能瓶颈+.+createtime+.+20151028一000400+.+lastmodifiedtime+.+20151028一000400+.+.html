not null字段, 说明字段内容不能为null.

否则字段为null时,为了说明这个字段不存储内容,会在字段后面添加扩展空间来标记字段是null的
因此在访问时, 会多去判断扩展空间内容, 造成效率降低

not null的使用:
创建时,使用not null说明字段非null
插入时,插入空值来替代null: insert into table values ('','');
搜索时,'is not null'搜索非null字段,'<> '''搜索非空值的字段.
