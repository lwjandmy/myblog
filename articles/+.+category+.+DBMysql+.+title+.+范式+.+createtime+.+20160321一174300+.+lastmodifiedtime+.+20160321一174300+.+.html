范式

1NF: 域只含原子值  记忆: 原子性
例:
班级(id, 人数)
班级(id, 男生数, 女生数)
不满足1NF, 人数不是原子值

2NF: 普通属性完全依赖主键  记忆: 完全依赖
例:
选课(课号, 学号, 成绩, 学生姓名)
完全依赖: (课号, 学号) -> 成绩
不完全依赖: (学号) -> 学生姓名

3NF: 普通属性不能传递依赖主键  记忆: 传递依赖
例:
学生(学号, 班号, 班名)
传递依赖: 学号 -> 班号 -> 班名

建立人属性表, 拥有姓名,朋友属性

drop table user;
create table user(
    id int primary key auto_increment not null,
    name char(50) not null,
    friend_of int not null
);

insert into user values(NULL, 'A', 0);
insert into user values(NULL, 'B', 1);
insert into user values(NULL, 'C', 1);

select * from user;

# 通过id查看用户信息
select * from user
where id = 1;

# 通过id查看用户的朋友列表
select * from user
where friend_of = 1;
