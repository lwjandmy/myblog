用const let 替代var

const定义常量, 定义时必须初始化, 如

const PI = 3.14;

let定义块级作用域的局部变量, 如

(function () {
    let i = 1;
    if (true) {
        let i = 0;
        console.log(i); // 输出0
    }
    console.log(i); // 输出1
})();

var的缺点是, 变量的作用域是函数域, 导致函数内部中块级作用域(if,while等)var变量定义互相干扰, 如

(function () {
    var i = 1;
    if (true) {
        var i = 0;
        console.log(i); // 输出0
    }
    console.log(i); // 输出0
})();
