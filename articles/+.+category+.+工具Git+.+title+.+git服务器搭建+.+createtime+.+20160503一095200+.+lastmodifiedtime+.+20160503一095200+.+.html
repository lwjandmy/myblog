git服务器搭建

首先需要服务器配置sshd

服务器:

sudo mkdir -p /opt/git         
sudo chown li /opt/git
cd /opt/git
mkdir repos1.git
git init --bare repos1.git          # 用git init --bare,就能在服务器上建立仓库

客户端:

git clone li@192.168.137.105:/opt/git/repos1          # 用ssh, 即可checkout仓库
touch data1
git add *
git commit -m 'add data1'
git push orign master
