函数陷阱

本地变量是静态检测的

X = 'global'    # X是全局变量
def fun():
    print(X)    # 正常, X是全局变量, 可以访问
def fun2():
    print(X)    # 出错, X变量没有初始化
    X = 'local' # X变量的初始化在这里

默认参数，可变对象

def fun(x=[]):     # x的默认值是列表类型，是可变对象
    x.append(1)    # 这里修改了x的默认值
    return x
fun()              # 返回[1]
fun()              # 返回[1,1]
# 修改方法:
def fun(x=None):
    if x is None:
        x = []     # 这里x获取了一个新的列表
    x.append(1)
