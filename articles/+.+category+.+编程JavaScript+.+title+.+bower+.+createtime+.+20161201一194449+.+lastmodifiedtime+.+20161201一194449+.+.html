bower

bower简明入门教程

bower解决js的依赖管理

查看命令列表

bower --help

查看命令帮助

bower install --help

初始化bower配置

bower init # 会生成bower.json配置文件

查看可下载的包

bower search jquery

查看包的可下载版本

bower info jquery

bower-doc-install

下载包并保存到bower.json

bower install jquery --save
bower install jquery#3.1.1 --save
bower install jquery#~3.1.1 --save
bower install jquery#^3.1.1 --save
bower install jquery#>=1.2.3 <2.0 --save
bower install jquery#<gh-pages> --save

查看项目中已导入的库

bower list

删除包

bower uninstall jquery

更新包版本

bower update jquery

更新bower.json中包版本

bower update

查看依赖树

bower list
