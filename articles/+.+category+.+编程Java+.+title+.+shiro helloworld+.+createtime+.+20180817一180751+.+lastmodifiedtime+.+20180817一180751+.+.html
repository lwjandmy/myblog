shiro helloworld

shiro框架内容

SubjectContext 创建Subject时的临时对象 里面就是个map 给Subject.Builder用来newSubjectContext, 给SecurityManager用来newSubject
    getSecurityManager(): SecurityManager
    getSessionId(): Serializable
    getSubject(): Subject
    getPrincipals(): PrincipalCollection
    getSession(): Session
    isAuthenticated(): bool
    getAuthenticationInfo(): AuthenticationInfo
SecurityManager
    login(Subject, AuthenticationToken): Subject
    logout(Subject): void
    createSubject(SubjectContext): Subject
SubjectFactory 
    createSubject(SubjectContext): Subject
SubjectDAO 
    save(Subject): Subject
    delete(Subject): void
Subject 当前运行用户的指定安全视图, 指当前与软件进行交互的物体
    getSession(): Session
    isAuthenticated(): bool
    login(AuthenticationToken): void
    hasRole(String role): bool
    checkPermissions(String ...) throw AuthorizationException
    getPrincipals(): PrincipalCollection
AuthenticationToken 一个认证凭证, 如用户名密码凭证, 指纹凭证, 生物识别凭证等
    getPrincipal(): Object 本金, 当事人, 身份标识, 如用户名, 身份证, 指纹摘要等
    getCredentials(): Object 文凭, 证明, 信任物, 身份校验, 如密码, 短信验证码等
AuthenticationInfo 代表一个用户的所有信息
    getPrincipal(): Object 本金, 当事人, 身份标识, 如用户名, 身份证, 指纹摘要等
    getCredentials(): Object 文凭, 证明, 信任物, 身份校验, 如密码, 短信验证码等
RememberMeManager "记住我"功能
    rememberSerializedIdentity(Subject) 记住
    forgetIdentity(Subject) 忘记
    getRememberedSerializedIdentity(SubjectContext)
SessionManager 会话容器
    start(SessionContext): Session 创建会话
    getSession(SessionKey): Session 获取会话
Authorizer 授权
    hasRole(PrincipalCollection, String role): bool
    checkPermission(PrincipalCollection, String permission) throw AuthorizationException
PrincipalCollection 原则 与Realm有什么关系?
    fromRealm(String realm): Collection
    getPrimaryPrincipal(): Object
    asList(): List
Authenticator.authenticate(AuthenticationToken) 认证
Realm 由Realm去解析AuthenticationToken, 得到AuthenticationInfo. Realm类似一个用户数据库, AuthenticationInfo是用户库的一条记录
    supports(AuthenticationToken): bool
    getAuthenticationInfo(AuthenticationToken):AuthenticationInfo
CacheManager.getCache 缓存管理器 获得缓存
EventBus 消息发送监听器
    publish(Object) 发布消息
    register(Object) 通过注解, 配置Listener, 通知Listener
    unregister(Object)
