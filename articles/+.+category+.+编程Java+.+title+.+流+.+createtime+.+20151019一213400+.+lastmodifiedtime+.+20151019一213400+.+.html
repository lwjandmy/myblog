读取字节: File -> FileInputStream (-> BufferedInputStream)
InputStream is = new BufferedInputStream(new FileInputStream(new File("test.dat")));

读取字符(1): File -> FileInputStream -> InputStreamReader (-> BufferedReader)
Reader reader = new BufferedReader(new InputStreamReader(new FileInputStream(new File("test.txt"))));
读取字符(2): File -> FileReader (-> BufferedReader)
Reader reader = new BufferedReader(new FileReader(new File("test.txt")));

字节流, 读取二进制字节, 类名以InputStream或OutputStream结尾
Object
|--InputStream()
|  |--FileInputStream(File)       *由File构造InputStream*
|  |--FilterInputStream(protected constructor)
|  |  |--BufferedInputStream(InputStream) 装饰器

字符流, 读取字符, 类名以Reader或Writer结尾
Object
|--Reader(protected constructor)
|  |--InputStreamReader(InputStream)        *由InputStream构造Reader*
|  |  |--FileReader(File)              *由File构造Reader*, 等于组合了FileInputStream和InputStreamReader
|  |--BufferedReader(Reader)  装饰器

字符转字节
byte[] byteArray = String::getBytes("UTF-8"); // 字符按照编码转换成字节

ANSI编码, 不是具体的某一种编码, 不同地区使用不同编码标准. 字符可能对应不定长的字节
UNICODE编码, 为各种语言的每个字符定义了唯一的编码, 因此可跨地区统一记录字符
