git-svn

在当前目录初始化svn仓库

git svn init .

从SVN仓库下载代码, 会创建test-repo目录

git svn clone --username=li_wjie https://localhost/svn/test-repo

提交代码

git svn dcommit

  1. 如果出错: is out of date

    1. 从仓库下载最新代码

      git svn rebase

    2. 如果出错: error: 无法合并变更

      1. 手动合并

        vim <conflict-file>

      2. 完成合并, 继续

        git add <conflict-file>
git rebase --continue

    3. 再次提交代码

      git svn dcommit

git svn info
