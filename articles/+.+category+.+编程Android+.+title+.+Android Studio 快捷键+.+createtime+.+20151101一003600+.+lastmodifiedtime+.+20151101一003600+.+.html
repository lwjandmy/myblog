Android Studio 快捷键

配置位置：File -> Settings -> Keymap

Ctrl+左键  定位资源
Shift+Enter  换行, 光标可以在一行中间
Ctrl+Alt+Enter  换行, 光标保留在上一行

Ctrl+Y 删除行
Ctrl+W 逐步按语义向外选择代码

Alt+/ 自动填充需要的import

Ctrl+D 复制当前行到下一行
Alt+Shift+Down 移动当前行到下一行

Alt+Enter 显示建议的代码. 出现类型需要转换等, 出现波浪线时, 用来弹出建议的代码

Ctrl+Alt+L  格式化代码  Code -> Reformat Code

Ctrl+Alt+T  Surround With...
