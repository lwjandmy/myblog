set declare export env shell定义变量

linux 中 set,env,export,declare显示shell变量的区别

环境变量的两种作用域

  1. 当前shell文件作用域 例:
    A1="1234"
  2. 用户环境变量作用域 例:
    export A1="1234"

env export

显示用户环境变量

例

env
export

set declare

显示用户环境变量和当前shell作用域的变量

例

set
declare

总结

只记住export一个就可以了, export可定义用户环境变量, 也可查看用户环境变量.

如果还要查看当前shell定义的变量, 用declare
