对于Java程序猿学习的建议

在校生:

  1. java基础教程 http://www.runoob.com/java/java-tutorial.html

    多动手

  2. HTML/CSS/JS 能写出简单页面, Servlet/JSP (J2EE) 需精通 多花精力 用eclipse 马士兵的视频, mysql&JDBC&navicat&sqlyog

  3. spring springmvc mytabis 做出增删改查Web项目, maven, http://edu.51cto.com/lesson/id-76468.html

  4. 找工作

工作1年内

  1. 1年内看完, "充分吸收", 在工作中实践

工作1-2年

  1. <设计模式>, 做笔记, 写博客

  2. 代码优化的书<重构 改善既有代码的设计>

  3. 能写出一手优雅的代码

工作2-3年

  1. <深入理解java虚拟机>

  2. 并发相关知识:

  3. 深入了解所用框架, 看jdk类源码, 框架源码

  4. 造轮子

工作3-4年

  1. 提升很难, 学习多样化: 大数据 分布式缓存 分布式消息服务 分布式计算 软负载均衡

  2. 建立差异性, 精通一项, 而不是肤浅的全会

  3. 成为优秀程序员

    前三年的学习内容, 只要是做过几年java开发都能掌握.

    需找个方向深入, 如: 分布式计算领域, 大数据领域, 并发领域.

  4. 弥补基础: 算法, 操作系统, 编译原理. <深入理解计算机系统>(适合后端开发), (适合网络编程), <数据结构与算法>(适合计算研究).

工作4-5年

  1. 遇到瓶颈, 深入下去收效甚微

  2. 提高影响力: 去知名公司, 发表文章, 创建开源项目. 能创造更多机会, 作用超过现在的技术能力.

  3. 参与到伟大项目中.
