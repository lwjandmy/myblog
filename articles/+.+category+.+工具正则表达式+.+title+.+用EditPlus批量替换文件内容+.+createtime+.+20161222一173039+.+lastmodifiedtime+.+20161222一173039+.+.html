用EditPlus批量替换文件内容

EditPlus -> 目录窗口 -> 找到项目所在文件夹 -> 右键 -> 在目录中查找

查找项: (\@Transactional.*\n[ \t]*)p[^u][^ \t]+
文件类型: *.java
文件夹: D:\workspace
勾选:使用正则表达式, 包含子目录

查找 -> 右键查找结果窗口 -> 全部打开 -> ctrl+h

查找项: (\@Transactional.*\n[ \t]*)p[^u][^ \t]+
替换项: \1public
勾选: 使用正则表达式, 到达文件末尾后从头开始, 所有打开文件

文件 -> 保存所有文件 -> 关闭所有文件.
