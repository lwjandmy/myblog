快捷键

运行Emacs Lisp命令

C-j 运行当前行的Lisp命令,执行结果显示在下一行,需要在lisp-interaction-mode中
C-x C-e 运行当前行的Lisp命令,执行结果显示在最下面的缓冲行,需要在lisp-interaction-mode中

.emacs配置文件

C:\Documents and Settings\Li\Application Data下，建立.emacs文件，设置配置信息，注意名字需要用cmd改

按字符,行(语言无关)调整插入符

C-b backward-char
C-f forward-char
C-p previous-line
C-n next-line

按词,句子,段落(语言相关)调整插入符

M-b
M-f
M-p
M-n

翻页

C-v scroll-up-command
M-v scroll-down-command

调整插入符所在行的显示位置

C-l recenter-top-bottom

到行(语言无关)头行尾 到句子(语言相关)头句子尾

C-a move-beginning-of-line
C-e move-end-of-line
M-a
M-e

到所有文字头尾

M-<
M->

重复操作

C-u n universal-argument

取消操作

C-g keyboard-quit

显示命令文档

C-h k describe-key

设置标记,用于选择文字

C-@ set-mark-command

移除选择区域

C-w kill-region

复制选择区域

M-w save the region, but don't kill it

移除一行

C-k kill-line

恢复移除的文字

C-y yanking(召回)

撤销

C-x u Undo

文件与缓冲

C-x C-f 寻找一个文件
C-x C-s 储存这个文件
C-x C-b 列出缓冲区
C-x b xx 进入xx文件的缓冲区
C-x s 保存多个缓冲区
C-x k 关闭缓冲区(关闭文件)

退出

C-x C-c 离开Emacs

查找与替换

M-x replace-string 替换
M-x text-mode Text模式
M-x auto fill mode 自动拆行,超过一行后,从空格位置拆行

多窗格操作

C-M-v 滚动下方窗格(向下)
C-M-S-v 滚动下方窗格(向上)
C-x o 切换到其它窗格
C-x 0 delete-window 关闭当前窗口
C-x 1 delete-other-windows 仅显示一个窗口
C-x 2 split-window-below 显示分窗口于下方
C-x 3 split-window-right 显示分窗口于右方
C-x 4 C-f 在其它窗口打开文件
C-x 4 C-o 在其它窗口切换缓冲区
