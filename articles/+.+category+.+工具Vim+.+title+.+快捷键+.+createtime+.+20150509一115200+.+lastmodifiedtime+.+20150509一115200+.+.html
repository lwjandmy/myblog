快捷键

:q    quit
:wq    write and quit

Ctrl-{        equal to ESC

i    input

h    left
l    right
j    up
k    down

0    line head
$    line tail
^    line head(not space word)

w    move a word
b    move back a word

e    move a word(ignore ,.:[]{})

(    移到当前句子开头，? . !结尾
)    移到下一个句子开头

{    移到当前段开头，段落是下一个空白行前的文本
}    移到下一段开头

[[    移到当前节的开头
]]    移到下一节开头

[N]X    do X N times

:set nu    set line number index

[N]G    goto index N line

a    append

c[w|b|0|$]    change [word|back-word|line-head|line-tail]
cc    change whole line
r    replace a word
[N]s    replace N word

d[w|b|0|$]    delete [word|back-word|line-head|line-tail]
dd    delete whole line
x    delete a word

u    undo (only undo last change)
"[N]p    undo index N record

y[w|b|0|$]    yank
p    paste

.    repeat

o    new line at next line
O    new line at previous line
+    move cursur to next line head
-    move cursor to prev line head





^F    next page
^B    prev page
^D    next half page
^U    prev half page



z[ENTER]    move screen, cursor at top
z.    move screen, cursor at middle
z-    move screen, cursor at bottom



/pattern    搜索  
?pattern    向前搜索
n        搜索下一个
N        搜索上一个

^G        显示当前编辑位置


vi -r [FILE]    恢复文件
:pre        强制写入缓冲区，用于恢复文件




v        进入选择模式
[d|c|y|v]i[w|s|p]    [删除|改变|复制|选择]i[词|句子|段落] 这里i代表inner,能自动找到词,句子等的开头,并选中它的inner部分
daw        和上面类似，删除一个词，不必考略当前插入符位置，它会自动找到词的开头，并选中它的all部分(all,包括周围空格;inner,不包括周围空格)



:colorscheme    设置颜色方案

:s/Old/New/    substitude 替换
:s/Old/New/g    substitude 替换,范围是整行
:%s/Old/New/g    substitude 替换,范围是整个文件
:%s/Old/New/gi    substitude 替换,范围是整个文件,忽略大小写
:6,8s/Old/New/g    substitude 替换,范围是6到8行
替换命令也可结合选择模式,选择要替换的范围
:%s//New/g    要替换的内容使用上次搜索的结果(/OR?)

补全命令：
C-x C-f 补全文件名
C-n / C-p 补全候选中切换

C-x C-n 向前搜索可补全内容

C-x C-d 用宏(define)补全
C-x C-v 用Vim命令补全

帮助技巧:

:help c_CTRL+D  在命令输入状态下, 按CTRL+D可查看相关命令

CTRL+] 进入链接
CTRL+O 退出链接


行内定位:

g0 gm g^ 移动到屏幕的行前/中/后, 在设置wrap line时有作用

3| 5| 移动到第几个字符

fx Fx 向右/左查找字符x

tx Tx 向右/左查找字符x的前一个字符(插入符在x前面)

3; 5; 再次执行3/5次f/t查找

3, 5, 再次执行3/5次f/t查找 按反方向



行间定位:

gj gk 向下/上移动, 在设置wrap line时有作用


单词定位:

w b 向前/后定位单词
e b 在某单词上定位首/尾字母


搜索:

* 对当前插入符所在单词进行搜索
g* 对当前插入符所在单词进行搜索 部分匹配搜索

gd gD 搜索当前插入符所在单词的本地/全局声明




CTRL-O 在插入模式中, 可临时切换到命令模式, 命令执行完后自动恢复插入模式

de 删除到单词最后一字符

s 把插入符所在字符删掉并进入插入模式

U/u 在选择模式下,把被选字符变大写/小写

CTRL-A CTRL-X 在插入符所在的数字中, 加/减1

>> << 向右/左平移TAB
