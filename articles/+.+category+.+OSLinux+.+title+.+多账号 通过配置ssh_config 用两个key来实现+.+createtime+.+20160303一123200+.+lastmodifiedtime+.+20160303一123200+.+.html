多账号 通过配置ssh_config 用两个key来实现

在GitHub多个帐号上添加SSH公钥

  1. ssh-keygen 执行key生成, 注意设置key保存位置为: /home/xxx/.ssh/id_rsa2

  2. 编辑~/.ssh/config, 添加

    Host github2
HostName github.com
User git
IdentifyFile ~/.ssh/id_rsa2.pub

    说明: ssh_config的语法是先设置一个Host, 用于'ssh github2'来代替'ssh git@github.com' HostName和User分别指明URL中各个变量名 IdentifyFile指明使用哪个pub key.
