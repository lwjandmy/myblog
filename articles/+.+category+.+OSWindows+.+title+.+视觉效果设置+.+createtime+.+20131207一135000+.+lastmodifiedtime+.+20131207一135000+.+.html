视觉效果设置

窗口内的动画控件和元素          // 在选择一些内容时, 淡入淡出选择项, 在输入印象笔记的时候, 有动画的感觉
启用透明玻璃                          // 窗口标题透明
启用桌面组合                          // 会影响到玻璃的其它效果
显示缩略图, 而不是显示图标    // 在设置桌面背景, 选择背景图片时, 可以看到图片预览图
在窗口和按钮上使用视觉样式   // 如果不选上, 会变成最原始的标准windows界面
在桌面上为图标标签使用阴影   // 桌面上图标标签加上阴影, 不加的话xp上会显示蓝色背景, vista里没有过度的感觉, 不好
