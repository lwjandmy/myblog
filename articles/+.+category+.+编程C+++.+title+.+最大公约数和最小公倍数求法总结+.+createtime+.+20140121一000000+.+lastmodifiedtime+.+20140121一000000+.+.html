最大公约数和最小公倍数求法总结

最大公约数和最小公倍数求法总结:


求2个数的最大公约数:
首先, 大数除以小数
然后, 除数除以余数
继续, 除数除以余数
当余数为0时, 除数即为最大公约数

原理:
a / b = c ... d
b / d = e ... 0
那么d整除b, b可表示为ed,
带入第一个式子
a / (ed) = c ... d
a = ced + d
因此d可整除a


求2个数的最小公倍数:
两数之积除以两数最大公约数即可

原理:
根据笔算最小公倍数
c    |    a    b
        d    e
那么a,b的最小公倍数为c * d * e
而a=c*d, b=c*e, 因此最小公倍数=a*b/c





由最大公约数(GreatestCommonDivisor)和最小公倍数(LeastCommonMultiple)的结合性, 可以得到求3个数的最大公约数(GreatestCommonDivisor)和最小公倍数(LeastCommonMultiple)的方法:


求3个数的最大公约数:
先求其中两个数的最大公约数, 用结果和第三个数再求一次最大公约数即可


求3个数的最小公倍数:
先求其中两个数的最小公倍数, 用结果和第三个数再求一次最小公倍数即可
