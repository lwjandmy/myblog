## cygwin乱码

编辑cygwin\Cygwin.bat:

```bash
@echo off

D:
chdir D:\cygwin64\bin

mintty zsh --login -i
```

注意用mintty替代原来的cmd


进入cygwin, 菜单 -> Options... -> Text -> Character set: UTF-8
