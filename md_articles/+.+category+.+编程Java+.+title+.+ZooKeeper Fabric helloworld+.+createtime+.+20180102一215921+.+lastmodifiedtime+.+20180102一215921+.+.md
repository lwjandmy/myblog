# ZooKeeper Fabric helloworld

进入客户端命令行

```bash
./bin/zkCli.sh -server 127.0.0.1:2181
```

查看帮助

```bash
help
```

查看zoo节点(znode)

```bash
ls /
```

创建节点/zk_test, 数据是my_data

```bash
create /zk_test my_data
```

读取节点数据

```bash
get /zk_test
```

写入节点数据

```bash
set /zk_test junk
```

删除节点数据

```bash
delete /zk_test
```


docker-compose配置

```yaml
networks:
  zookeeper0:
  zookeeper1:
  zookeeper2:
services:
  zookeeper0:
    container_name: zookeeper0
    image: hyperledger/fabric-zookeeper:x86_64-1.0.3
    restart: always
    hostname: zookeeper0
    ports:
      - "2181:2181"
      - "2888:2888"
      - "3888:3888"
    environment:
      - ZOO_TICK_TIME=5000
      - ZOO_MY_ID=1
      - ZOO_SERVERS=server.1=zookeeper0:2888:3888 server.2=zookeeper1:2888:3888 server.3=zookeeper2:2888:3888
    networks:
      - zookeeper0
```
