# gdb debug helper function

查看QString的调试辅助函数

```java
const char *d_QString(const QString &str)
{
    static QByteArray ba;
    ba = str.toLatin1();
    return ba.constData();
}
```

```java
const char *d_QStringRef(const QStringRef &str)
{
    return d_QString(*str.string());
}
```
