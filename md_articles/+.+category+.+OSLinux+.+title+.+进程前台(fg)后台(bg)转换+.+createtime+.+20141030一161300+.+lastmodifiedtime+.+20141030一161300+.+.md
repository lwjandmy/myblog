# 进程前台(fg)后台(bg)转换

程序启动时的前台后台设置:
```bash
sleep 1     # 前台运行程序
sleep 1 &   # 后台运行程序
```

程序运行中的前台后台设置:
```bash
Ctrl-Z      # 程序后台暂停
jobs        # 查看当前shell有哪些后台任务, 可得到任务id(job id, GID)
bg GID      # 将后台暂停的任务继续后台执行
fg GID      # 将后台暂停的任务前台继续执行
Ctrl-C      # 终止前台程序
```

管理全部终端的程序的暂停/继续/退出:
```bash
ps aux      # 查看当前程序列表, 可得到程序id(process id, PID)
kill -stop PID  # 暂停某程序
kill -cont PID  # 继续运行某程序
kill PID    # 终止某程序
kill -kill PID  # 由内核终止某程序, 用于某程序停止响应, 无法正常关闭
kill -l     # 查看可发送的信号列表
```
