# sudo配置


编辑配置文件:

```bash
visudo -f /etc/sudoers
```

修改配置, 允许sodu组用户获得sudo权限:

```ini
visudo -f /etc/sudoers # 配置sudo用户组
%sudo ALL=(ALL) ALL # 去掉注释, 应用这句配置
%sudo ALL=(ALL) NOPASSWD: ALL # 或者使用这个配置, 无需输入密码
```

创建sudo组:

```bash
groupadd sudo
```

加入用户到sudo组:

```bash
usermod li -G sudo
```
