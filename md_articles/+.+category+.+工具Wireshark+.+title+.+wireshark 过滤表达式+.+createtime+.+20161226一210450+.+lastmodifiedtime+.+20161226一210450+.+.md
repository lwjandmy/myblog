# wireshark 过滤表达式


## 限制ip

```text
ip.addr == 192.168.32.31
```

## 限制端口

```text
tcp.srcport == 80 && tcp.dstport == 80
```

## 限制http网页

```text
http && ip.addr == 192.168.32.31
```

## 限制请求uri

```text
http && ip.addr == 192.168.32.31 && http.request.uri contains ajax
```
