# archlinux 安装 X server


安装Xserver

```bash
pacman -S xorg-server
```

安装XFree86驱动vesa

```bash
pacman -S xf86-video-vesa
```

生成xorg.conf配置文件:

```bash
X -configure
```

保存配置文件:

```bash
mv /root/xorg.conf.new /etc/X11/xorg.conf
```

启动Xserver:

```bash
X &
```

Ctrl+Alt+F1切换回终端
