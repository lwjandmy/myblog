## 镜像与容器

镜像, 一个只读的文件

容器, 是从某镜像为基础创建的运行实例, 运行实例不会修改镜像数据, 修改操作会保存在镜像之上的可写层, 可写层最终会变成镜像, 变为只读.

------

查找镜像

```bash
docker search tutorial
docker search httpd
```

列出本地镜像

```bash
docker images
```

下载镜像

```bash
docker pull learn/tutorial
docker pull ubuntu:13.10
```

在镜像中执行命令

```bash
docker run learn/tutorial echo "hello word"
```

查看镜像状态

```bash
docker ps
```

查看镜像资源使用率(类似top)

```bash
docker stats
```

关闭镜像

```bash
docker stop
```

删除镜像

```bash
docker rm
```

暂停镜像

```bash
docker pause
```
