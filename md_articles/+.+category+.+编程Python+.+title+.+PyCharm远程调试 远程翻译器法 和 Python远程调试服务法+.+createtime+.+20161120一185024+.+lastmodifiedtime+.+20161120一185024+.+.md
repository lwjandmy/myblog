

# PyCharm远程调试 远程翻译器法 和 Python远程调试服务法

## 远程翻译器法

PyCharm -> File -> Settings -> Project Interpreter -> Add Remote -> SSH Credentials

填写ssh信息

这种方式源码必须在本地, 不能再远程
