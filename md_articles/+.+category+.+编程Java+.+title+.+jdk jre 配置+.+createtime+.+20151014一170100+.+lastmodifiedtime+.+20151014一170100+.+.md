# jdk jre 配置

```text
JAVA_HOME  
E:\Java\jdk1.7.0

PATH 加入  
%JAVA_HOME%\bin;

类搜索路径  
CLASSPATH
.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\rt.jar;%JAVA_HOME%\lib\tools.jar

dt.jar  DesignTime 指明开发环境如何显示Java组件, 包含Swing类
rt.jar  run time 运行时需要的类, 在执行java命令时,需要这些包
tools.jar  tools 需要编译一个类的时候需要的, 执行javac时需要
```
