# cout 输出格式调整 格式备份恢复

格式调整

```c++
#include <iomanip>
    cout << "输出十六进制, 大写字母" << endl;
    cout << hex << uppercase << 123 << endl;
    
    cout << "保留2位小数" << endl;
    cout << fixed << setprecision(2) << M_PI << endl;
    
    cout << "设置输出宽度5, 左对齐, 用*号填充" << endl;
    cout << setw(5) << left << setfill('*') << 1 << endl;
```


格式备份恢复

```c++
ios state(0);
state.copyfmt(cout); // 备份
cout.copyfmt(state); // 恢复
```
