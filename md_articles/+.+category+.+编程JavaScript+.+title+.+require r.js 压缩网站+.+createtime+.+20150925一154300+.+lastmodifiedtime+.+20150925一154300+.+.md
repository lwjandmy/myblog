# require r.js 压缩网站

使用r.js压缩网站：
```js
node tools/r.js -o tools/build.js
```


当前r.js所在文件结构：
```text
./www     网站根目录
./www/app    网站js脚本，main.js等
./www/lib      网站要用到的模块
./tools/r.js     r.js脚本位置
./tools/build.js    r.js的配置文件
```



build.js:
```js
{
    "appDir": "../www",     
    "mainConfigFile": "../www/app.js",   
    "dir": "../www-built",      
    "modules": [             
        {
            "name": "app",
            "include": ["jquery", "app/base"]
        }
        {
            "name": "app2",
            "include": ["jquery2", "app/base2"]
        }
    ]
}
```
