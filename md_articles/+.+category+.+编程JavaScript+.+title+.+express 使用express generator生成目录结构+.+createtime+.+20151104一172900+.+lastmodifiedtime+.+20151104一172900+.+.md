# express 使用express generator生成目录结构

express 使用express generator生成目录结构

安装express generator:

```bash
npm install express-generator -g
```

创建learnexpress应用:
```bash
express learnexpress
```

安装依赖包:

进入learnexpress/目录
```bash
npm install
```

启动应用:
```bash
set DEBUG=learnexpress & npm start
```

访问网站:

http://localhost:3000
