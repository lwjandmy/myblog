# adb命令

进入fastboot
```shell
adb reboot bootloader
```

解除oem锁定
```shell
fastboot oem unlock     adb版本：2.0.0.0
```

刷入recovery
```shell
fastboot flash recovery recovery.img 
```

手动进入第三方recovery   
关机状态下,音量下键+开机键

关闭adb服务
```shell
adb kill-server
```

连接设备
```shell
adb devices
```
