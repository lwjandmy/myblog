# thinkphp 模板 display show 模板赋值assign

## 用模板文件来输出

```php
display('[模板文件]'[,'字符编码'][,'输出类型'])
```

用字符串来输出

```php
show('渲染内容'[,'字符编码'][,'输出类型'])
```

给模板赋值

```php
$this->assign('name',$value);
$this->assign(array('name' => 'thinkphp', 'email' => 'test@test.test'); 用数组批量模板赋值
```
