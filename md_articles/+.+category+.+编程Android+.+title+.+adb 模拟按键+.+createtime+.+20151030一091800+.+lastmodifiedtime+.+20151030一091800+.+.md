# adb 模拟按键

[转](http://blog.csdn.net/jlminghui/article/details/39268419)

按了设备的Backkey键
```bash
adb shell input keyevent 4 
```

解锁屏幕
```bash
adb shell input keyevent  82 
```

做划屏操作，前四个数为坐标点，后面是滑动的时间（单位毫秒）
```bash
adb shell input swipe 50 250 250 250 500 
```

点击坐标点x=50  y=250的位置
```bash
adb shell input tap 50 250 
```

输入字符abc
```bash
adb shell input text abc
```
