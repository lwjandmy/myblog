# nc Net Cat 网络数据传输


nc, Net Cat, 网络工具, 使用TCP,UDP,UNIX-domain套接字, 执行一次发送或接收命令

```bash
# 监听, 端口8080
nc -l 8080

# 建立连接, ip和port
nc localhost 8080

# 向url发送请求
printf "GET / HTTP/1.1\r\n\r\n" | nc baidu.com 80

# 端口扫描, 超时时间2秒
nc -w 2 -z baidu.com 20-30
echo "QUIT" | nc -w 2 baidu.com 20-30
```
