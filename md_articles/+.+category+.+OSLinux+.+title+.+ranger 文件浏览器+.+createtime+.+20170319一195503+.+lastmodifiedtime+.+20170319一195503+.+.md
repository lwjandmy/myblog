# ranger 文件浏览器


## 安装

cygwin:

```bash
cd ~
git clone --depth 1 https://github.com/ranger/ranger
cd ranger
make install
cd ..
yes | rm -r ranger
```

## 使用

```bash
# 在当前路径下进入shell
快捷键 S

# 查看有哪些快捷键
?k

# 进入home路径 goto home 
gh

# 进入root路径 goto /
gr

# 进入/etc路径l
ge

# 查看目录占用空间
du
```
